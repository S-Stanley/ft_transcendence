# Secrets

Plusieurs variables secrètes sont necessaires pour faire fonctionner
le projet et doivent se trouver a la racine des projets concernées dans
un `.env`:

### API

* POSTGRES_USER
* POSTGRES_DB
* POSTGRES_PORT
* CLIENT_ID
* CLIENT_SECRET

### WEBAPP

* REACT_APP_CLIENT_ID
* REACT_APP_REDIRECT_URI
